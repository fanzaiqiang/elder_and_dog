### Slide 4: 系統架構圖

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   使用者     │     │   AI 大腦   │     │   機器狗    │
│  「找水瓶」  │ ──▶ │  Claude/GPT │ ──▶ │    Go2     │
└─────────────┘     └──────┬──────┘     └─────────────┘
                           │
                    ┌──────▼──────┐
                    │  MCP 協定   │
                    │  ros-mcp    │
                    └──────┬──────┘
                           │
                    ┌──────▼──────┐
                    │ GPU Server  │
                    │ YOLO + DA3  │
                    └─────────────┘
```

**關鍵技術：**
- **MCP (Model Context Protocol)**：LLM 直接控制 ROS2
- **YOLO-World**：開放詞彙物件偵測
- **Depth Anything V2**：深度估計（距離判斷）

---

### Slide 5: 技術創新 - 「熟食策略」

| 傳統方案 | 我們的方案 |
|---------|-----------|
| 給 LLM 看圖片 | 給 LLM 看 JSON |
| 延遲 3-7 秒 | **延遲 1.5-2.5 秒** |
| 不可控 | 可調試、可解釋 |

**「不給 LLM 吃生肉，給它吃熟食」**

```json
{
  "label": "water bottle",
  "distance": "1.2m",
  "direction": "左側",
  "cmd_vel": {"linear_x": 0.3, "angular_z": 0.2}
}
```

---

### Slide 6: MCP 控制鏈

```
使用者 → LLM → MCP → rosbridge → ROS2 → WebRTC → Go2
```

**首次將 Anthropic MCP 協定應用於四足機器人！**

可用 MCP Tools：
| Tool | 功能 |
|------|------|
| `find_object` | 搜尋指定物件 |
| `go2_perform_action` | 執行動作（Hello, Dance 等）|
| `check_gpu_server` | 檢查 GPU 伺服器狀態 |
| `list_go2_actions` | 列出可用動作 |

---

### Slide 7: 可用動作展示

**Go2 支援 24 種動作：**

| 類別 | 動作 |
|------|------|
| 🤝 互動 | Hello, FingerHeart, WiggleHips |
| 💃 舞蹈 | Dance1, Dance2 |
| 🐕 基礎 | StandUp, Sit, Stretch |
| ⚡ 進階 | FrontFlip, MoonWalk（危險勿試！）|
