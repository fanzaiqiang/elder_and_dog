# Kilo Code 測試腳本

**用途：** 驗證 MCP + GPU Server + Go2 整合功能  
**前置條件：** GPU Server 已啟動、Go2 已連線

---

## 🧪 測試 1：GPU Server 健康檢查

請輸入以下指令給 Kilo Code：

```
檢查 GPU Server 是否正常運作
```

**預期結果：** 回覆 GPU Server 狀態 OK，版本 2.0.0

---

## 🧪 測試 2：尋找椅子

請輸入：

```
幫我找椅子
```

**預期結果：**
- 拍照
- 呼叫 find_object API
- 回報椅子位置（距離、方向）

---

## 🧪 測試 3：Go2 動作

請輸入：

```
向我打招呼
```

**預期結果：** Go2 執行 Hello 動作

---

## 🧪 測試 4：完整流程

請輸入：

```
找到椅子後，走過去然後打招呼
```

**預期結果：**
1. 找到椅子
2. 移動靠近
3. 執行 Hello 動作

---

## 🛠️ 故障排除

| 問題 | 解法 |
|------|------|
| GPU Server 無回應 | 確認 SSH Tunnel 建立 |
| 找不到物件 | 確認相機對準目標 |
| 動作沒反應 | 確認 Go2 連線正常 |
