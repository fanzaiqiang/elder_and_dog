## 🔍 必須先釐清的 5 大問題（按優先順序）

### 問題 1：感測器資料品質

**現況：**
- `/point_cloud2` 曾出現 `width=0`、`row_step=0`（01-16 已修復）
- `/scan` 曾全為 `inf`，修復後頻率約 5-7 Hz

**驗證指令：**
```bash
# 檢查 PointCloud2 資料結構
ros2 topic echo /point_cloud2 --once | grep -E "width|height|row_step"

# 檢查 /scan 有效點比例（排除 inf）
ros2 topic echo /scan --once | grep -v "inf" | wc -l

# 檢查 /scan 頻率
ros2 topic hz /scan
```

**優化目標：**
- `/scan` 頻率 ≥ 7 Hz 穩定
- 有效點比例 ≥ 80%

---

### 問題 2：里程計漂移程度

**現況：**
- 來自 Go2 內部融合（IMU + 輪子里程計）
- 01-17 曾出現未發布問題（未設定 `ROBOT_IP`）

**驗證指令：**
```bash
# 檢查 /odom 頻率
ros2 topic hz /odom

# 檢查 TF 連續性（map -> base_link）
ros2 run tf2_ros tf2_echo map base_link
```

**優化目標：**
- 短程（10m）累積誤差 < 10%
- TF 無斷裂（連續輸出）

---

### 問題 3：地圖更新機制

**現況：**
- `/map` 大多為 `-1`（未知區域）
- SLAM 可能沒吃到 `/scan` 或 TF/里程計資料不完整

**驗證指令：**
```bash
# 檢查 SLAM 反饋
ros2 topic echo /slam_toolbox/feedback --once

# 檢查 /map 資料內容（前 500 字元）
ros2 topic echo /map --once | grep -E "^data:" | head -c 500

# 檢查地圖解析度
ros2 topic echo /map --once | grep -E "resolution|width|height"
```

**優化目標：**
- 旋轉後 `/map` 應出現 `0`（free）和 `100`（occupied）
- 地圖更新延遲 < 3 秒

---

### 問題 4：TF 座標系一致性

**現況配置：**
- `odom_frame: odom`
- `map_frame: map`
- `base_frame: base_link`

**驗證指令：**
```bash
# 生成 TF 樹狀圖
ros2 run tf2_tools view_frames

# 檢查各感測器的 frame_id
ros2 topic echo /scan --once | grep frame_id
ros2 topic echo /point_cloud2 --once | grep frame_id
ros2 topic echo /odom --once | grep frame_id
```

**優化目標：**
- TF 樹完整無斷鏈
- 所有 frame 可正確轉換到 `base_link`

---

### 問題 5：動態環境處理（四足特有）

**Go2 特有問題：**
- 腳會被 LiDAR 掃到（產生噪點）
- 行走時機體搖晃（影響點雲品質）

**驗證方法：**
```bash
# 靜止時測量
# 停止機器狗，記錄 /scan 有效點比例
ros2 topic echo /scan --once | grep -v "inf" | wc -l

# 行走時測量
# 控制機器狗移動，再次記錄 /scan 有效點比例
```

**優化目標：**
- 行走時 `/scan` 仍有 ≥ 70% 有效點
- 可考慮引入高度過濾（過濾掉腳部點雲）

## 📝 理論背景：SLAM 7 步驟流程

```
1. 感測器輸入 (Sensor Input)
   ↓ LiDAR / Camera / IMU
2. 特徵擷取 (Feature Extraction)
   ↓ 角點 / 邊緣 / SIFT / ORB
3. 特徵匹配 (Feature Matching)
   ↓ ICP / 描述子匹配
4. 姿態估計 (Pose Estimation / Odometry)
   ↓ EKF / 粒子濾波器
5. 迴圈閉合 (Loop Closure)
   ↓ 識別已訪問位置
6. 束調整 (Bundle Adjustment)
   ↓ 非線性優化
7. 地圖輸出 (Map Output)
   ↓ 佔柵格地圖 / 特徵地圖
```
